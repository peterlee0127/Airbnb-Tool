# Airbnb-Tool
A easy tool to download airbnb data.

<h3>Step</h3>
1: Install node.js	
2: $ npm install		
<hr>
<h3>Search by Location</h3>
search.js	

<code>$ node search 'location' startPage endPage</code>		
<code>$ node search Tokyo 1 200</code>

<hr>


all the data will post to server which defined at data.js
